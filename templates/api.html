API Documentation

Access posts, authors, tags, and categories via JSON endpoints.

{% include breadcrumbs/single.html %}

📝 Posts

🔹 GET /api/posts.json

Returns all posts (non-paginated).

[
  {
    "id": "my-post-title",
    "title": "My Post Title",
    "date": "2025-09-15",
    "url": "/2025/09/15/my-post-title",
    "excerpt": "This post is about...",
    "categories": ["tech"],
    "tags": ["jekyll", "api"],
    "author": "john-doe"
  }
]

🔹 GET /api/posts/:slug.json

Returns full content and metadata for a specific post.

{
  "id": "my-post-title",
  "title": "My Post Title",
  "date": "2025-09-15",
  "url": "/2025/09/15/my-post-title",
  "excerpt": "This post is about...",
  "categories": ["tech"],
  "tags": ["jekyll", "api"],
  "author": "john-doe",
  "content": "
Full HTML content...
"
}

👤 Authors

🔹 GET /api/authors.json

List all authors, including post count and profile link.

[
  {
    "id": "john-doe",
    "name": "John Doe",
    "bio": "A mysterious writer",
    "website": "https://johndoe.com",
    "count": 4,
    "url": "/api/authors/john-doe.json"
  }
]

🔹 GET /api/authors/:id.json

Returns author info and their posts.

{
  "id": "john-doe",
  "name": "John Doe",
  "bio": "A mysterious writer",
  "website": "https://johndoe.com",
  "count": 4,
  "posts": [
    {
      "title": "Intro to Ruby",
      "date": "2025-09-14",
      "url": "/2025/09/14/intro-to-ruby",
      "excerpt": "A short guide...",
      "categories": ["tech"],
      "tags": ["ruby"]
    }
  ]
}

🏷 Tags

🔹 GET /api/tags.json

List all tags with post counts and URLs.

[
  {
    "id": "jekyll",
    "slug": "jekyll",
    "count": 3,
    "url": "/api/tags/jekyll.json"
  }
]

🔹 GET /api/tags/:id.json

Returns all posts with the given tag.

{
  "id": "jekyll",
  "count": 3,
  "posts": [
    {
      "title": "Building a Jekyll Plugin",
      "date": "2025-09-12",
      "url": "/2025/09/12/jekyll-plugin",
      "excerpt": "Learn how to build plugins...",
      "categories": ["dev"],
      "author": "john-doe"
    }
  ]
}

📂 Categories

🔹 GET /api/categories.json

List all categories with post counts and links.

[
  {
    "id": "Tech",
    "slug": "tech",
    "count": 5,
    "url": "/api/categories/tech.json"
  }
]

🔹 GET /api/categories/:id.json

Returns all posts in the given category.

{
  "id": "Tech",
  "count": 5,
  "posts": [
    {
      "title": "Intro to Ruby",
      "date": "2025-09-14",
      "url": "/2025/09/14/intro-to-ruby",
      "excerpt": "A short guide...",
      "tags": ["ruby"],
      "author": "john-doe"
    }
  ]
}
